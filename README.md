## Build and run

Please fill a grid like this in `grid.txt`.

```
-------------------------
| 0 7 0 | 0 6 0 | 8 0 0 | 
| 0 3 0 | 0 0 0 | 0 6 0 | 
| 0 0 0 | 0 0 0 | 0 2 7 | 
-------------------------
| 0 2 9 | 0 5 0 | 0 1 0 | 
| 0 0 4 | 0 3 0 | 7 8 0 | 
| 0 0 0 | 0 0 6 | 5 0 4 | 
-------------------------
| 3 0 0 | 2 0 0 | 0 0 0 | 
| 0 0 6 | 0 8 3 | 0 0 0 | 
| 0 0 2 | 0 0 9 | 0 7 0 | 
-------------------------
```

To generate an empty grid:
```shell
./empty_grid > grid.txt
```


Linux:

```shell
./sudoku_solver
```
